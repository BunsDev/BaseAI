## Getting started with `@baseai/core` SDK

### Installation

First, install the `@baseai/core` package using npm, pnpm, or yarn:

```bash
npm install @baseai/core
```

or

```bash
pnpm add @baseai/core
```

or

```bash
yarn add @baseai/core
```
